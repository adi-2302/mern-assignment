Introduction

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications. It describes the structure of a web page semantically and originally included cues for the appearance of the document.

History

In 1980, physicist Tim Berners-Lee, a contractor at

CERN

European Organization for Nuclear Research, known as CERN, is an intergovernmental organization that operates the largest particle physics lab in the world. Established in 1954, it is based in Geneva, Switzerland.

, proposed a system for sharing documents over the internet.

He later wrote the first web browser and server, launching the World Wide Web project at CERN. This became the basis for modern web technology.

Markup

HTML uses various tags to define elements like <h1>, <p>, <a>, <div>, etc. These tags are used to structure the content of the page.

Example Code

Below is a basic example of an HTML document:


<!DOCTYPE html>
<html>
  <head>
    <title>This is a title</title>
  </head>
  <body>
    <div>
      <p>Hello world!</p>
    </div>
  </body>
</html>
